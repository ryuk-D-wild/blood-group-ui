# Blood Group UI

A modern user interface for managing and exploring blood group information. This project provides an intuitive and responsive frontend designed to help users view, filter, and interact with blood group data.

## Features

- View various blood groups and their compatibility information
- Search and filter records efficiently
- Responsive design for desktop and mobile devices
- User-friendly navigation
- Clean and modern UI components

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- [npm](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ryuk-D-wild/blood-group-ui.git
   ```
2. Navigate to the project directory:
   ```bash
   cd blood-group-ui
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

### Running the App

Start the development server:

```bash
npm start
```
or
```bash
yarn start
```
Visit [http://localhost:3000](http://localhost:3000) to view it in your browser.
